# Staticflowchamber
COMSOL-MATLAB LiveLink scripts for parametric CFD simulations optimizing static flux chamber geometry for methane emission detection from abandoned oil and gas wells. Includes single-parameter sweeps (Phase 1) and Latin Hypercube Sampling (Phase 2) across seven design parameters.
I have added codes and datasets used in Time series prediction of the collected live data here as well.
